Linguaggio per la gestione di un ricettario
Software per una facile e versatile gestione di un ricettario
Esame di Linguaggi e Modelli Computazionali LS
Abstract
Il progetto riguarda lo sviluppo di un software che metta ordine in una ricetta, la cataloghi e la renda disponibile in varie visualizzazioni (per aiutare l'utente nella preparazione).
Specifiche
Linguaggio di Programmazione Java 6
Generazione automatica di lexer e parser
JavaCC 5.0 & J.T.B. 1.3.2
Ambiente di sviluppo principale: Eclipse 3.6 (Helios) + Ant + plugin JavaCC
Ambiente di sviluppo GUI: NetBeans 7
Altre librerie:
  • Jgraph per graficare una ricetta
  • MySql-Connector per la connessione a database remoti
